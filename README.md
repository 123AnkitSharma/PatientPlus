# PatientPlus

Healthcare management system with patient, doctor, and admin portals.

## Features

- Patient portal for booking appointments
- Doctor portal for managing appointments
- Admin portal for system management

## Tech Stack

- React.js
- Node.js + Express.js
- MongoDB
- Razorpay payment integration

## Setup

1. Clone the repository
2. Install dependencies

```bash
cd backend && npm install
cd ../frontend && npm install
cd ../admin && npm install
```

3. Create `.env` files in each directory.
4. Run the application

```bash
# Backend
cd backend && npm start

# Frontend
cd frontend && npm run dev

# Admin
cd admin && npm run dev
```
